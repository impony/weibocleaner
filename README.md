# 微博环卫工

微博环卫工是 Opera 桌面浏览器的一款插件，可以将新浪微博网页版的广告、活动、推介等灰尘全部扫走，还你一个清爽的阅读环境，不管是 Presto 还是 Chromium 内核均可安装，在插件设置菜单中，还可以设置微博页面配色和宽度。安装后访问新浪微博 http://weibo.com ，登录后就能看到效果了。

## 插件成品地址 ##
